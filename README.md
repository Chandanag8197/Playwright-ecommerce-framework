# Playwright E-Commerce Automation Framework

## Overview
Production-grade E2E automation framework built using Playwright + TypeScript.

## Features
- UI Automation
- TypeScript support
- HTML reporting
- GitHub Actions ready
- Scalable structure

## Tech Stack
- Playwright
- TypeScript
- Node.js

## How to Run

Install dependencies:
npm install

Run tests:
npx playwright test

View report:
npx playwright show-report
